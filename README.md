# Store_Platform

>FrontEnd part of the site is an online store. The following pages/certain parts of the page have been completely laid out:
- main page
- product page
- footer
- header
- basket
- favorites
- catalog
- privacy policy
- error page

**Libraries**:
- gulp
- node js

To install libraries with `npm install` use:
```
npm install --global gulp-cli
```
